- The "raw data" folder includes csv files containing the raw data collected from a person with aphasia
at five assessment time points as well as difference scores associated with two tDCS treatments.
"data_singlecol.csv" contains all raw data, while the other files contain different parts of these
overall data for subanalyses.

- "tdcs_analysis.R" file contains the R code for the descriptive (mean and median) and inferential 
(Friedman, Wilcoxon) analyses of the data.
